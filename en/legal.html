4. Resonance Counter (GoatCounter)

For the statistical evaluation of visitor resonance, this website uses the privacy-friendly service GoatCounter (Martin Tournoij, Ireland). GoatCounter uses no cookies and stores no personal data (such as names, device IDs, or precise locations). Furthermore, no data is shared with third parties.

When the resonance button is clicked, GoatCounter creates a cryptographic hash using the IP address, user agent, and a rotating "salt" (a random value). This hash is stored on servers in Germany or Finland and allows GoatCounter to detect if the same user clicks multiple times within a few hours (preventing artificial inflation of the counter), without ever identifying the user. After a maximum of 8 hours, the "salt" is deleted, rendering the hash completely useless—making re-identification impossible.

For more information, please refer to the GoatCounter Privacy Policy. Browser detection has been disabled in the settings. Therefore, the dashboard only shows the website operator how many users from which country clicked the button.

The legal basis for this processing is a legitimate interest in reach measurement in accordance with Art. 6(1)(f) GDPR.
